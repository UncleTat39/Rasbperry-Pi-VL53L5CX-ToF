# Rasbperry-Pi-VL53L5CX-ToF


Rasbperry Pi  use VL53L5CX ToF to measure the distance.


**Use code for watch this map**
          GPIO map
VL53L5CX Pin |   Rasbperry Pi GPIO Pin
VCC          |   3.3V
GND          |   GND
SDA          |   SDA1 (GPIO 2)
SCL          |   SCL1 (GPIO 5)
LPN          |   GPIO 17
